# Student Experiences In Academic Environments
## Overview
This repository contains the published HTML version of my PSY 120L final reproducible
report.
This project examined the effects of Autonomy-Supportive Academic Feedback and Controlling Academic Feedback on Self-Efficacy and Achievement Motivation, and whether or not Perceived Parental Warmth influences these outcomes.
## Report
The published report is available through GitHub Pages.
## Tools
The report was created using Quarto and R.
## Files
This repository includes:
- `index.html`: the published HTML version of the reproducible report
- `README.md`: a brief description of the repository
## Note
The dataset is not included in this repository. Only the rendered HTML report is published.
